# 🏸 羽毛球运动功能助手

> 羽毛球技术动作指导小程序，涵盖球拍参数对比、握拍方式详解、技术分类训练。

## 功能介绍

| 页面 | 说明 |
|------|------|
| 🏸 球拍详情 | 查看球拍参数（重量、平衡点、拉线磅数等） |
| ⚖️ 球拍对比 | 多支球拍参数横向对比，辅助选购决策 |
| 🤝 握拍详解 | 正手/反手/网前/杀球握拍方式图解 |
| 📚 技术分类 | 各项羽毛球技术动作分类学习 |

## 快速开始

### 1. 环境准备

下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

### 2. 导入项目

1. 打开微信开发者工具，点击「导入项目」
2. 选择本目录
3. 填入 AppID（可使用测试号）
4. 点击「导入」即可预览
